Brand Identity

Waymark is a travel-tracking app built for digital nomads and expats. Our design system reflects a travel-editorial, modern aesthetic with a deep-green brand identity, saffron accents, and warm parchment surfaces.

Personality: Empowering, trustworthy, minimal, editorial.

Usage Guidelines

Color Usage

✓ Do
Use brand color for primary CTAs and headlines. Use accent for highlights and secondary actions.
✗ Don't
Don't use brand and accent together on the same element. Avoid low-contrast text combinations.

Spacing

✓ Do
Use consistent spacing from the scale. Create visual hierarchy with spacing rather than color.
✗ Don't
Don't use arbitrary spacing values. Don't overcrowd elements—embrace whitespace.

Typography

✓ Do
Use weight and size to create hierarchy. Keep line length under 80 characters for body text.
✗ Don't
Don't stack too many type sizes. Avoid all-caps body text; use for labels only.

Dark Mode

All design tokens automatically adapt to dark mode via CSS custom properties. Test designs in both modes.
